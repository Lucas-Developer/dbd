DBD - DNS Block Dodge

Configuração automática de servidores de DNS.

View on GitHub Download .zip Download .tar.gz

O que é?

O DBD é um pequeno e simples script que altera os DNS do computador para os DNS públicos da Google. Deixando assim de ser necessário usar os DNS do ISP, consequentemente, deixando de estar obrigado aos bloqueios idiotas por partes destes.

P.S.: Não, não estamos a chamar idiotas aos ISP's. Sim, estamos a chamar idiotas às organizações que, de certa forma, obrigaram os ISP's a imporem o bloqueio.

Para quem é?

O DBD está (e estará) apenas disponível para Windows e MacOS. Destina-se aos meros mortais, vulgos utilizadores comuns, que não se querem aventurar a fazer alterações no seu computador que exijam mais de dois cliques.

Instruções

Windows

  • Descarregar ZIP
  • Descompactar
  • Executar winconfig.vbs
  • Aproveitar

MacOS

  • Descarregar ZIP
  • Descompactar
  • Botão direito no ficheiro macconfig.app
  • Carregar em "Abrir"
  • Selecionar Wi-Fi or Ethernet e pressionar Ok
  • Escrever a password
  • Aproveitar

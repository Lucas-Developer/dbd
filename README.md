# Auto DNS Config

## Windows Instructions
- Download ZIP
- Extact it
- Execute winconfig.vbs
- Enjoy 

## Mac Instructions
- Download ZIP
- Extact it
- Open Terminal
- Change into extracted directory
- Write "chmod u+x macconfig.sh", without quotes
- Write "./macconfig.sh"
- Choose 1 for Wi-Fi or 2 for Ethernet
- Write your password
- Enjoy
